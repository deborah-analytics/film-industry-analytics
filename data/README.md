## Dataset Information

The datasets used in this project are not included in the repository due to size limitations.

### Data Source

The data was obtained from the publicly available IMDb datasets:
https://datasets.imdbws.com/

### Data Structure

The datasets are provided as tab-separated values (TSV) files containing structured information about films, television titles, and associated metadata.

### Key Datasets Used

* **title.basics**: Contains title information such as genre, release year, and runtime
* **title.ratings**: Contains average ratings and vote counts
* **title.akas**: Contains alternative titles and regional releases
* **name.basics**: Contains information about cast and crew members
* **title.principals**: Contains relationships between titles and cast/crew roles

### Usage in the Project

The datasets were used to:

* integrate multiple data sources into a unified dataset
* analyse relationships between genres, ratings, and runtime
* explore cast and crew characteristics
* support data-driven insights into film and television success

### Note

To reproduce this analysis, the datasets can be downloaded directly from the IMDb website and processed using the provided pipeline.
